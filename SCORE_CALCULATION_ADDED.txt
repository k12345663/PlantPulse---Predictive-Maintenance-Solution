FORMULA 1: Anomaly Score
```
Anomaly Score = 2^(-E(h(x)) / c(n))

Where:
- E(h(x)) = Average path length to isolate point x
- c(n) = Normalization factor
- n = Number of samples
```

FORMULA 2: Normalization Factor
```
c(n) = 2 * H(n-1) - (2(n-1)/n)

Where:
- H(n-1) = ln(n-1) + 0.5772 (Euler's constant)
- n = Number of samples (10 machines)

Example for n=10:
c(10) = 3.748
```

FORMULA 3: Confidence Score
```
confidence = min(100, int(abs(raw_score) * 100))

Example:
raw_score = -0.87
confidence = 87%
```

  📈 EXAMPLE CALCULATION WALKTHROUGH
═══════════════════════════════════════════════════════════════════════════

MACHINE M6 (Anomalous):

Step 1: Extract Features
```
incident_count: 33
avg_downtime: 45.5
temp_fix_ratio: 0.36
critical_ratio: 0.45
recent_incidents: 5
issue_diversity: 4
```

Step 2: Normalize Features
```
incident_count: (33-20)/8 = 1.625
avg_downtime: (45.5-35)/10 = 1.05
temp_fix_ratio: (0.36-0.25)/0.10 = 1.1
critical_ratio: (0.45-0.30)/0.12 = 1.25
recent_incidents: (5-2)/1.5 = 2.0
issue_diversity: (4-3)/1 = 1.0
```

Step 3: Isolation Forest Processing
```
100 trees isolate M6 at different path lengths
Average path length E(h(M6)) = 2.8
```

Step 4: Calculate Score
```
c(10) = 3.748
Anomaly Score = 2^(-2.8/3.748) = 0.595
Raw score from scikit-learn = -0.87
```

Step 5: Determine Prediction
```
raw_score (-0.87) < threshold
Prediction = -1 (ANOMALY)
```

Step 6: Calculate Confidence
```
confidence = abs(-0.87) * 100 = 87%
```

RESULT:
```python
{
    'is_anomaly': True,
    'anomaly_score': 0.87,
    'confidence': 87,
    'reason': 'Unusual pattern detected'
}
```
