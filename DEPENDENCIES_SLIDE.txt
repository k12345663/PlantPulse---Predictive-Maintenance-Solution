CORE DEPENDENCIES (requirements.txt):

1. streamlit >= 1.28.0
   Purpose: Web application framework
   Why: Interactive UI, real-time updates, easy deployment

2. pandas >= 2.0.0
   Purpose: Data manipulation and analysis
   Why: DataFrame operations, CSV handling, data processing

3. numpy >= 1.24.0
   Purpose: Numerical computing
   Why: Array operations, mathematical calculations

4. scikit-learn >= 1.3.0
   Purpose: Machine learning algorithms
   Why: Isolation Forest, StandardScaler, ML training

5. plotly >= 5.17.0
   Purpose: Interactive visualizations
   Why: Charts, graphs, 3D heatmaps, interactive plots

6. python-dotenv >= 1.0.0
   Purpose: Environment variable management
   Why: Configuration, API keys, settings

7. openai >= 1.3.0
   Purpose: OpenAI API integration (optional)
   Why: Alternative LLM provider (cloud-based)

8. google-generativeai >= 0.3.0
   Purpose: Google Gemini API integration (optional)
   Why: Alternative LLM provider (cloud-based)

9. chromadb >= 0.4.0
   Purpose: Vector database (optional)
   Why: Future enhancement for semantic search

═══════════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Common Issues:

1. Streamlit not found
   Solution: pip install streamlit

2. Ollama connection error
   Solution: Check if Ollama is running
   Command: ollama list

3. Database locked
   Solution: Close other connections
   Or: Restart application

4. Port already in use
   Solution: streamlit run app.py --server.port 8502

5. Module import errors
   Solution: pip install -r requirements.txt
